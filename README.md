# HTML-CSS

This repository contains all the files I created while learning **HTML and CSS**. It includes basic concepts, practice exercises, and a few small projects that I built to understand how things work in real scenarios.

I’ve organized the files topic-wise so it’s easier to go through different concepts step by step.

## What’s inside

* HTML basics (forms, semantic tags, inline/block elements, etc.)
* CSS fundamentals (selectors, colors, fonts, box model)
* Layout techniques (Flexbox, Grid, positioning)
* Responsive design using media queries
* CSS animations, transitions, and transforms
* Practice exercises and small design challenges
* A Netflix clone UI project and some random experiments

## Why I made this

I created this repository mainly to practice and keep track of what I’ve learned. Instead of just watching tutorials, I tried to write code myself and experiment with different ideas.

## Notes

* Some parts are based on tutorials, but I’ve tried to modify and practice them on my own
* Not everything is perfect — this is part of my learning process
* The structure may not be very polished, but it reflects my progress over time
